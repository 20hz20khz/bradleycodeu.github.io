AP CS PRINCIPLES

What Will You Learn in APCSP?

AP Computer Science Principles is an introductory course that doesn't require any previous computer programming knowledge. This course will use Scratch, Python, Javascript, and Java. We will cover primitive data types, lists/arrays, dictionaries, user input, functions, conditionals, loops, reading/writing to files, and handling errors. We will touch on GUIs and classes, but we dive deeper into those in our advanced courses. We will use a project-based approach in which students spend the majority of class time working on diverse projects that demonstrate their understanding of given structures.

GAME DESIGN

What Will You Learn in Game Design?

Game Design is an advanced course with a prerequisite of APCSP (or Programming, the previous intro course). In this course you will learn Object-oriented Programming using Python, P5.js, Node.js, and Unity with C#.
